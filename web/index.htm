G O T I T   


Gene, Occurence and Taxa in Integrative Taxonomy

Use this page to login and access to GOTIT web user interface

Welcome to the demo version of GOTIT, a database structure and web interface for optimizing the input, management, and sharing of data, metadata and vouchers produced on a day-by-day basis by biodiversity laboratories involved in the delimitation, inventory and distribution of species. The tool is particularly suitable for designing and monitoring biodiversity projects that employ an integrative taxonomic approach combining morphology-based and DNA-based species occurrence data to explain biodiversity patterns. Benchmark data in the demo version are not to be used for publication.

You can upload here the help documentation to use GOTIT web interface

Login to the demo version is granted under two different roles.
  • Collaboration members can create, update, and delete their own data but they only have a read access to others’ users data.
  • Invited members only have read access to the data..

Login details are:
  • Collaboration members : Username: cm; Password: cm-GOTIT
  • Invited members : Username: im; Password: im-GOTIT
